var, let, and const

var: var is a function & global scoop. It can be redeclared & reassign anywhere of the code. It hoisting behavior is initialized with undefined.

let: let is a block scoop. It can't be redeclared but can be reassign anywhere of the code. It hoisting behavior is uninitialized.

const: const is a block scoop. It can't be redeclared & reassign anywhere of the code. It hoisting behavior is uninitialized.



null & undefined

If we declared a variable but could not define any value inside of variable. For this, result is undefined. Null is another thing that we can define a variable as null like let a = null; It means inside of a variable there is no value. So, undefined and null little difference is that we use null directly when we don't want to put value inside of a variable but we can't use undefined directly.
